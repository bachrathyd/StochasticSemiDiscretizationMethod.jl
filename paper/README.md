# Accompanying paper

- [`main_MFSSD.pdf`](main_MFSSD.pdf) — *Multiplication-Free Stochastic
  Semi-Discretization for the Efficient Moment Stability Analysis of Vibration
  Systems with Delay*, D. Bachrathy and H. T. Sykora (submitted to the
  *Journal of Sound and Vibration*).
- [`cover_letter.pdf`](cover_letter.pdf) — submission cover letter.

The LaTeX sources are kept separately; only the compiled PDFs are included here.

## Reproducing the figures

Every figure is generated by a script in [`../benchmark`](../benchmark). Set that
folder up once (from the repository root):

```julia
julia --project=benchmark -e 'using Pkg; Pkg.develop(path="."); Pkg.instantiate()'
```

| Figure / result | Script(s) |
|-----------------|-----------|
| Work-precision & memory scaling (MF vs explicit product vs copy-recursion) | `benchmark_wp_ultra.jl`, `benchmark_wp_baseline.jl`, `plot_wp_ultra_pub.jl`, `combined_wp.jl` |
| Factored vs dense operator | `benchmark_factored_vs_dense.jl` |
| MF complexity sweep | `benchmark_mf_complexity.jl` |
| GPU scaling (chain of oscillators, CPU vs GPU) | `benchmark_cpu_gpu.jl`, `benchmark_cpu_gpu_wp.jl`, `benchmark_chain_dof.jl`, `plot_chain_gpu_pub.jl`, `colab_cpu_gpu.ipynb` |
| Beam modal-convergence study | `benchmark_beam_fe.jl`, `plot_beam_fe_pub.jl` |
| 2-DOF SSV milling stability & quality chart | `ssv2dof_chart.jl` (variance colormap + stochastic boundaries), `ssv2_det_curves.jl` (deterministic boundaries), `plot_ssv2.jl` (render) |
| SSV Monte-Carlo validation | `ssv2_mc_validation.jl` |

The final data behind each chart is stored next to the scripts as `.csv`, so the
`plot_*` scripts reproduce the figures in seconds without re-running the
(hours-long) sweeps.
